## 1. What is an Arrow Function

An arrow function is a shorter syntax for writing functions in
JavaScript.

Example:

``` javascript
const add = (a, b) => a + b;
console.log(add(2,3));
```

Arrow functions do not have their own `this` binding and are often used
for short functions.

------------------------------------------------------------------------

## 2. What are Promises

A Promise represents a value that may be available now, later, or never.

It has three states: - Pending - Resolved (fulfilled) - Rejected

Example:

``` javascript
const promise = new Promise((resolve, reject) => {
  resolve("Success");
});
```

------------------------------------------------------------------------

## 3. Need of Promises Even After Callbacks

Callbacks can lead to **callback hell** (nested callbacks).

Promises provide: - Better readability - Error handling using
`.catch()` - Chaining using `.then()`

Example:

``` javascript
fetch(url)
  .then(res => res.json())
  .catch(err => console.log(err));
```

## 8. Can We Declare `const` Without Initialization

No.

`const` variables must be initialized at the time of declaration.

Invalid:

``` javascript
const a;
```

Valid:

``` javascript
const a = 10;
```

## 9. Need of Anonymous Functions

Anonymous functions are functions without a name.

They are useful when: - Passing functions as arguments - Using
callbacks - Using array methods

Example:

``` javascript
setTimeout(function(){
 console.log("Hello");
},1000);
```

------------------------------------------------------------------------

## 10. Block Scope in JavaScript

Block scope means a variable is only accessible inside the block `{}`
where it is declared.

Variables declared with: - `let` - `const`

Example:

``` javascript
if(true){
  let x = 10;
}
```

`x` cannot be accessed outside the block.

## 12. `this` Keyword in JavaScript

`this` refers to the object that is currently executing the function.

Example:

``` javascript
const person = {
  name: "Abhay",
  greet: function(){
    console.log(this.name);
  }
};
```

------------------------------------------------------------------------

## 13. Can We Access Variable Outside the Function

Variables declared inside a function cannot be accessed outside.

Example:

``` javascript
function test(){
  let x = 10;
}
```

`x` is not accessible outside the function.

------------------------------------------------------------------------

## 14. Find Length of an Array

Use the `.length` property.

Example:

``` javascript
const arr = [1,2,3,4];
console.log(arr.length);
```

------------------------------------------------------------------------

## 15. What Happens When We Don't Return in Function

If a function does not return anything, JavaScript automatically returns
`undefined`.

Example:

``` javascript
function test(){
  console.log("Hello");
}
```

Output when called:

    undefined

## 16. Common Case for Naming Variables

Common naming conventions:

1.  camelCase → `firstName`
2.  PascalCase → `UserName`
3.  snake_case → `user_name`
4.  UPPER_CASE → `MAX_VALUE`

Most commonly used in JavaScript is **camelCase**.
