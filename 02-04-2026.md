# Web Development Interview Questions (02-04-2026)

## 1. What is a Web Application?

A web application is a software program that runs on a web server and is
accessed through a web browser over the internet. Examples include
Gmail, Facebook, and online shopping platforms.

------------------------------------------------------------------------

## 2. How can you give styling in JavaScript?

You can apply styles in JavaScript using: -
`element.style.property = "value"` - Adding/removing CSS classes using
`classList`

Example:

``` js
element.style.color = "red";
element.classList.add("active");
```

------------------------------------------------------------------------

## 3. Common Types of API Requests

-   GET → Fetch data
-   POST → Create data
-   PUT → Update data
-   DELETE → Remove data

------------------------------------------------------------------------

## 4. What is Django and why do we need it?

Django is a high-level Python web framework used to build secure and
scalable web applications quickly.

Need of Django: - Fast development - Built-in authentication - ORM for
database handling - Security features

------------------------------------------------------------------------

## 5. In a normal scenario, who initiates requests?

In most cases, the client (browser or mobile app) initiates the request
to the server.

------------------------------------------------------------------------

## 6. Difference between HTTP and HTTPS

  -----------------------------------------------------------------------
  Feature                   HTTP                HTTPS
  ------------------------- ------------------- -------------------------
  Security                  Not secure          Secure (encrypted)

  Protocol                  HyperText Transfer  HyperText Transfer
                            Protocol            Protocol Secure

  Port                      80                  443

  Data Encryption           No                  Yes (SSL/TLS)
  -----------------------------------------------------------------------

------------------------------------------------------------------------

## 7. Difference between Stateful and Stateless APIs

  Feature      Stateful                     Stateless
  ------------ ---------------------------- -----------------
  Memory       Server stores client state   No state stored
  Complexity   More complex                 Simpler
  Example      Sessions                     REST APIs

------------------------------------------------------------------------

## 8. How do you remove the last child in JavaScript?

``` js
parentElement.removeChild(parentElement.lastElementChild);
```

## 10. Explain Closure

A closure is when a function remembers and can access variables from its
outer scope even after the outer function has finished executing.

------------------------------------------------------------------------
