## 1. Difference between Inline and Block Elements

**Block Elements** - Start on a new line. - Take the full width
available. - Can contain other block and inline elements.

**Examples** - `<div>` - `<p>` - `<h1>` to `<h6>` - `<section>`

**Inline Elements** - Do not start on a new line. - Take only the width
required by the content. - Usually used inside block elements.

**Examples** - `<span>` - `<a>` - `<strong>` - `<em>`

## 2. Why we use `DEFAULT` keyword in SQL

The `DEFAULT` keyword assigns a default value to a column when no value
is provided during insertion.

``` sql
CREATE TABLE students (
  id INT,
  name VARCHAR(50),
  country VARCHAR(50) DEFAULT 'India'
);
```

## 3. Use of Lambda Function

A lambda function is a small anonymous function used for short
operations.

``` python
square = lambda x: x * x
print(square(5))
```

  Command    Description
  ---------- ------------------------------------
  DELETE     Removes specific rows using WHERE
  TRUNCATE   Removes all rows quickly

## 7. What is `map()` Function

`map()` applies a function to every element of an iterable.

``` python
numbers = [1,2,3,4]
result = list(map(lambda x: x*2, numbers))
print(result)
```

------------------------------------------------------------------------

## 8. What is Foreign Key

A Foreign Key links two tables and maintains referential integrity.

``` sql
CREATE TABLE orders(
  order_id INT,
  customer_id INT,
  FOREIGN KEY (customer_id) REFERENCES customers(id)
);
```

------------------------------------------------------------------------

## 9. zip()` Function

`zip()` combines elements from multiple lists.

``` python
names = ["A", "B", "C"]
scores = [80, 90, 75]

print(list(zip(names, scores)))
```

------------------------------------------------------------------------

## 10. What is `enumerate()` in Python

`enumerate()` adds an index to iterable items.

``` python
fruits = ["apple", "banana", "mango"]

for index, fruit in enumerate(fruits):
    print(index, fruit)
```
